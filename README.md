# 🚀 Smart Network Monitoring System

💻 A Python-based system for real-time network traffic monitoring and anomaly detection.

---

## 📌 Overview

This project is designed to monitor network traffic, analyze packets, and detect suspicious activities in real-time. It helps improve network security by identifying unusual patterns and generating alerts.

---

## ⚡ Features

- 📡 Real-time packet sniffing  
- 🧠 Anomaly detection system  
- 📊 Traffic analysis  
- 📧 Email alert system  
- 🖥️ Simple dashboard interface  

---

## 🛠️ Tech Stack

- Python 🐍  
- Scapy (for packet analysis)  
- Streamlit , Pandas ,  Plotly
- SQ lite 
- SMTP (for email alerts)  
- Basic UI (Dashboard)

## ▶️ How to Run

1. Clone the repository:
```
git clone https://github.com/hsybmalik96/Smart-Network-Monitoring-System.git
```

2. Navigate to project folder:
```
cd Smart-Network-Monitoring-System
```

3. Install dependencies:
```
pip install scapy
```

4. Run the project:
```
python main.py
```

---

## ⚠️ Important Notes

- Run with administrator/root privileges for packet sniffing  
- Configure email credentials in `email_alerts.py`
